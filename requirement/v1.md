# QuickPG 项目开发记录 (V1 - MVP 阶段)

## 1. 项目愿景
- **定位**：一款专为 macOS 打造的轻量、极简、高性能 PostgreSQL 客户端。
- **目标**：解决现有免费工具“太重、网页感强、内存占用高”的痛点。
- **发布计划**：前期自用打磨，后期开源发布。

## 2. 当前技术栈
- **核心框架**：Electron + React + TypeScript
- **构建工具**：electron-vite (极速编译与热重载)
- **UI 方案**：Tailwind CSS v4 (实现 macOS 原生质感、毛玻璃效果)
- **编辑器**：Monaco Editor (VS Code 同款，支持 SQL 高亮与快捷键)
- **数据库驱动**：`pg` (node-postgres)
- **图标库**：Lucide React
- **测试框架**：Vitest + React Testing Library (实现单元测试与组件交互测试)

## 3. 已实现功能 (V1 MVP)
### 核心交互
- [x] **macOS 原生 UI**：无边框设计、 traffic-light 适配、侧边栏半透明效果。
- [x] **多窗口会话管理 (Tab Mode)**： (已添加测试)
    - 支持同时连接多个数据库，像浏览器一样切换 Tab。
    - 每个 Tab 拥有独立的连接上下文、SQL 编辑器和数据视图。
- [x] **自动化测试 (QA)**：
    - 集成 Vitest 单元测试框架，覆盖核心业务链路。
- [x] **单窗口多任务逻辑**：左侧连接/表导航，右侧工作区。 (已添加测试)
- [x] **快捷键支持**：`Cmd+Enter` 快速执行 SQL。 (已添加测试)

### 数据库操作
- [x] **多连接并发支持**：后端重构为连接池映射（Connection Pool Map），支持多个库同时在线。 (已添加测试)
- [x] **连接管理**：支持 Host, Port, User, Password, Database 基础连接配置。 (已添加测试)
- [x] **智能导航**：连接后自动抓取 Schema 和 Table 列表。 (已添加测试)
- [x] **SQL 执行器**：集成 Monaco Editor，支持任意 SQL 编写与执行。 (已添加测试)
- [x] **可编辑数据网格**： (已添加测试)
    - 自动识别表主键 (Primary Key)。
    - **双击单元格直接修改数据**，实时同步至数据库。
- [x] **表结构管理 (Structure Mode)**： (已添加测试)
    - 查看字段详情（类型、是否为空、默认值）。
    - **直接在界面修改字段名、修改字段类型**。

## 4. 运行与打包指令
- **开发调试**：`npm run dev`
- **运行测试**：`npm test`
- **本地编译**：`npm run build`
- **macOS 打包**：`npm run package:mac` (生成 .dmg)

## 5. 待办事项 (Backlog)
- [ ] **持久化存储**：本地保存连接配置（目前刷新后需重新输入）。
- [ ] **密码安全**：对保存的密码进行加密处理。
- [ ] **性能优化**：引入虚拟滚动 (Virtual List) 处理万级以上的数据展示。
- [ ] **交互增强**：右键菜单支持（删除表、导出 SQL 等）。
- [ ] **UI 细节**：应用图标设计、连接状态提示。
